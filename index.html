Prototype Flow (Wiring)

High-level wiring between screens for demo flows.

  1. Login (00) → Executive (03) via Demo Login (Option B)
  2. HR (04) manages candidates → Committee (05) review
  3. Finance (06) receives handoffs from HR/Exec
  4. Payroll (08) runs monthly exports to Reports (09)
  5. Audit (07) records all handoffs and approvals
  6. Admin (10) adjusts settings, global components (02) and design tokens (01)
